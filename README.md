# SimulFeel
Interfaces with Buttplug/Intiface to simulate the intimacies of a human.
## Introduction
Instead of creating scripts and running them I wanted to change it up.  Why do we need scripts if we aren't going to sync them to a video?  What I wanted to do is simple, create profiles that send certain commands repetitively.
## Requirements/Supported Hardware
Designed to work with devices that are Linear (stroke/oscillate)
  - Windows 10 w/Bluetooth adapter
  - Buttplug/Intiface client
  - Devices
     - Kiiroo Onyx 2
     - Fleshlight Launch
